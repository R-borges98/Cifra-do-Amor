​# Cifra do Amor 🔐

Bem-vindo ao *Cifra do Amor*, uma aplicação web desenvolvida como um exercício de lógica e controle de fluxo.

## 🚀 Sobre o Projeto
O *Cifra do Amor* é uma aplicação web focada em lógica de acesso e experiência do usuário. O objetivo principal é simular uma barreira de autenticação, onde o usuário precisa inserir a senha correta para prosseguir. Este projeto é de minha autoria e foi desenvolvido como uma homenagem pessoal, inspirado pela leitura do livro Fortaleza Digital.

> Nota: Este projeto foi desenvolvido para fins didáticos, focando na lógica de fluxo de navegação e na construção de interfaces interativas.

## 🛠️ Tecnologias Utilizadas
* *HTML5*: Estruturação semântica de todo o conteúdo.
* *CSS3: Estilização com animações inspiradas no universo *Matrix e design totalmente responsivo para dispositivos móveis.
* *JavaScript (Vanilla)*: Implementação da lógica de autenticação (bloqueio de rotas não autorizadas) e manipulação dinâmica do DOM.
